Formato del fichero

El fichero Excel debe tener estas columnas en la primera fila:

ColumnaEjemplo
ProfesorGarcía López, María
DiaLunes / Martes / Miércoles...
Franja1ª hora / 2ª hora / Recreo...
Tipoclase / libre / reunion
AsignaturaMatemáticas (opcional)

Valores válidos para "Tipo":

  • clase — tiene clase, no puede hacer guardia
  • libre — franja libre, disponible para guardia
  • reunion — ocupado, no puede hacer guardia
